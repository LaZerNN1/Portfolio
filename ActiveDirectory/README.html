Oppsummering

  • Opprettet VM (DC01) med 4 GB RAM og 4 CPU-kjerner i VirtualBox
  • Installert Windows Server 2022 med Bridged Adapter-nettverk
  • Promotert serveren til Domain Controller for lab.local
  • Konfigurert DNS for domenet
  • Opprettet OU-er (Brukere og Groups) i Active Directory
  • Opprettet brukere: John Doe og Alice Smith
  • Opprettet grupper: HR og IT (sikkerhetsgrupper)
  • Lagt John Doe til som medlem av HR-gruppen
  • Konfigurert Group Policy med passordbegrensninger
